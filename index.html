ApproachPřístup

How I actually workJak to doopravdy dělám

01

Problem firstNejdřív problém

I start with what's missing or broken. That's the first impulse for building an app. Začínám tím, co chybí nebo nefunguje. To je první impuls pro tvorbu aplikace.

02

Draft in an afternoon. Production-ready in days.Náčrt za odpoledne. Do produkce během pár dní.

Getting to a working prototype takes one afternoon. Getting it right — auth, error handling, edge cases — takes a bit longer. Both steps happen. K funkčnímu prototypu se dostanu za jedno odpoledne. Než je to správně — přihlášení, ošetření chyb, okrajové případy — chvilku to trvá. Obě fáze ale proběhnou.

03

Security and compliance built inBezpečnost a compliance už od začátku

Every project goes through a security checklist before launch. GDPR, cookie consent, RLS policies. Part of the process from day one. Každý projekt před spuštěním projde bezpečnostním checklistem. GDPR, souhlas s cookies, RLS politiky. Součást procesu od prvního dne.

04

Open by defaultOtevřeně jako výchozí stav

The code is on GitHub. Read it, fork it, or tell me what's wrong with it. Kód je na GitHubu. Můžeš si ho přečíst, forknout, nebo mi říct, co je špatně.
